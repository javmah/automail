AutoMail is an Email automation, Email Notification & Auto-reply plugins, It works with WordPress, WooCommerce, Contact form seven, and other Plugins.

## üí° Description ##

Email automation, Email Notification & Auto-reply plugins. WordPress platform along with its plugin generates many events 
and event base data, those data are vital for day-to-day operations. This plugin sends an email notification with generated 
data when an event occurs. This plugin supports around all WordPress events along with many third-party plugins. The list 
of events are:

## üîù  Email Automation Events are ##

1. [Contact form 7](https://wordpress.org/plugins/contact-form-7/) forms submission to automatic email notification.

2. WordPress New user registration to automatic email notification & Auto-reply.

3. WordPress  User login & logout to automatic email notification & Auto-reply.

4. WordPress User Profile Updates to automatic email notification & Auto-reply.

5. WordPress User Login to automatic email notification & Auto-reply.

6. WordPress User Logout to automatic email notification & Auto-reply.

7. WordPress Delete User to automatic email notification & Auto-reply.

8. WordPress New post to automatic email notification & Auto-reply.

9. WordPress update post to automatic email notification & Auto-reply.

10. WordPress New page automatic email notification & Auto-reply.

11. WordPress New Comment to automatic email notification & Auto-reply.

12. WordPress Edit Comment to automatic email notification & Auto-reply.

13. [Gravity Forms ](https://www.gravityforms.com/) to automatic email notification & Auto-reply.

14. [ninja form](https://wordpress.org/plugins/ninja-forms/) to automatic email notification & Auto-reply.

15. [Formidable form](https://wordpress.org/plugins/formidable/) to automatic email notification & Auto-reply.

16. [WPform](https://wordpress.org/plugins/wpforms-lite/) submission to automatic email notification & Auto-reply.

17. [weForms](https://wordpress.org/plugins/weforms/) submission  to automatic email notification & Auto-reply.

18. custom post type events [PRO] to automatic email notification.

19. [WooCommerce [PRO]](https://wordpress.org/plugins/woocommerce/) New order to automatic email notification & Auto-reply.

20. [WooCommerce [PRO]](https://wordpress.org/plugins/woocommerce/) Order status change to automatic email notification & Auto-reply.

21. [WooCommerce [PRO]](https://wordpress.org/plugins/woocommerce/) new product to automatic email notification & Auto-reply.

22. [WooCommerce [PRO]](https://wordpress.org/plugins/woocommerce/) Update product to automatic email notification & Auto-reply.

23. [ACF- Advanced Custom Fields [PRO]](https://wordpress.org/plugins/advanced-custom-fields/)

= Modern Way: =

2. Search For "automail".
3. Install, then Activate it.
4. Follow the step-by-step documentation on the settings page

= Old Way: =
1. Upload `automail` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the step-by-step documentation on settings page
